We put together this website for an assignment for an NJIT graduate course: IS 601 (Web Systems Development). This course is about learning a variety of things that have to do with current software technologies, developing applications with Python and MySQL, learning about the fundamental concepts that deal with the Internet and programming, and collaborating with source code.

For this assignment, we have created a website dedicated to the History of the Internet. This includes the following:

  • Invention of the Internet
  • The Browser Wars
  • The Internet Wars
  • Dot Com Bubble
  • The People
